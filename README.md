# storage-api-test

Test project
Cloud api storage

Для запуска:
1. Клонируем репозиторий (git clone https://github.com/bggshe4ka/storage-api-test)
2. Запускаем docker командой ./vendor/bin/sail up или sail up
3. Выставляем права на файлы
4. composer install
5. npm install
6. npm run build
7. php artisan migrate
